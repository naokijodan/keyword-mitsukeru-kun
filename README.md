# キーワードみつける君

商品リサーチ中にキーワードを視覚的に管理できるChrome拡張機能です。

## 機能

- **注目キーワード（緑）** - 登録したキーワードが緑でハイライト
- **未分類キーワード（黄）** - 新しいキーワードが黄色で表示、クリックで振り分け
- **除外機能** - 除外キーワードを含むタイトル全体をグレー表示

## インストール

1. [Chrome Web Store](#)からインストール
2. または、このリポジトリをダウンロードして`chrome://extensions`から読み込み

## 使い方

1. 商品リサーチページを開く
2. 黄色いキーワードをクリック
3. 「注目に追加」または「除外に追加」を選択

---

## Privacy Policy

### 収集するデータ
この拡張機能は個人情報を収集しません。

### 保存するデータ
- ユーザーが登録した注目キーワード
- ユーザーが登録した除外キーワード

これらのデータはブラウザのローカルストレージにのみ保存され、外部サーバーには送信されません。

### 第三者への提供
データを第三者に販売、提供、転送することはありません。
